Project Proposal

VIRTUAL ZOO

  1. Purpose: The purpose of my web application is to allow people to experience the thrill of a zoo without have to go to a zoo or pay for it.
  2. Audience: This is people who are not able to go to a zoo or who want to experience the zoo within the confines of their own home.
  3. Data Sources:
    • I will be using the Heroku Zoo Animal External API.
    • I will also be using localStorage to store comments.
    • Local JSON file for slideshow pictures, sponsors, links to surveillance animal videos.
  4. Initial Module list:
    • Slide show for home page
    • List of Animals with pictures from API
    • Comment Section
    • Donation Form
  5. Wireframes: Link to wireframe pages
    • Home Page
    • Multiple Animals Page
    • Individual Animal Pages
    • About Us Page
    • Donate Page
  6. Colors/Typography/Specific Element Styling
    • Typography: Zen Tokyo Zoo for headers, Gill Sans for body
    • Header and Footer Background Color:
      #002400
    • Main Background Color:
      #DCE0D9
    • Navbar:
      #7B904B
    • Animal Details Background:
      #58641D
    • Form Background:
      #273B09
    • Animation on some buttons (ex. submit button)
    • Animation on Thank You Page
